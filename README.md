### Dependencies

For gradle:

In build.gradle add:

```json
dependencies {
    implementation 'com.google.code.gson:gson:2.8.5'
}
```

For native java:

Download dependency jar and add to project.

[GSON 2.8.5](https://github.com/google/gson)
